AppKit Ethers VanillaJS dApp Example

This is a simple example of how to use the AppKit Ethers library in a vanilla JavaScript dApp. It demonstrates connecting to a wallet, signing messages, sending transactions, and more.

Note: This example uses the AppKit Ethers library, which is designed to work with the Reown AppKit. Make sure you have the Reown AppKit installed and configured in your wallet.
